# Northcoders News

Northcoders news is a full stack reddit-style aggregation board. The Front End is built using React.js, HTML and CSS. A link to the back end git hub repository can be seen below. Northcoders news allows the user to view articles, filter the articles and vote on them. As well as being able to post comments.

- Deployed version: https://nc-news-jpreynolds.netlify.app
- Back End: https://github.com/JPReynolds/be-nc-news.git

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Make sure you have installed all of the following prerequisites on your development machine:

- Node.js v13.8.0 - [Download & Install Node.js](https://nodejs.org/en/download/) and the npm package manager.

### Installing

Clone the repository in the terminal using:

```
git clone https://github.com/JPReynolds/fe-nc-news.git

cd fe-nc-news
```

Install dependencies:

```
npm install
```

To run the application, use the following command. This will open the browser and run the application using localhost on port 3000.

```
npm start
```
